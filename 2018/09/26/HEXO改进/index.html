发表于 2018-09-26 | 分类于 基本设置

如何改变皮肤？

步骤如下：

  1. 前往 hexo网站 选择皮肤，并点击进入github页面；
  2. 将git库clone到themes文件夹下；
  3. 修改conf配置文件，改为对应的名字；
  4. 有的图片需要配置按照github的readme来写。

如何自定义博客结构？

  1. 进入使用的themes下，修改layout中的相关文件（比如js这些）
